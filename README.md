## 📌 About the Project  
This project helps job-seekers instantly improve their resumes using AI.

## ✨ Key Features  
- AI skill extraction  
- ATS scoring  
- Missing skills detection  
- Job description matching  
- Export final report to PDF  
- Fully responsive UI  
- FastAPI backend  

## 🧰 Tech Stack  
### Frontend  
- React  
- Vite  

### Backend  
- Python  
- FastAPI  

## 🔧 Installation  
### Backend  
\\\ash

python -m venv venv

\\\
